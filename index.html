    How to Use the Family Tree App

    1. Use the form to add family members by providing their name, relationship, and type of relation.
    2. Select "View Family Tree" to visualize the relationships.
    3. You can export the family tree as a JSON file using "Export JSON".
    4. Use "Import JSON" to upload an existing family tree.
    5. To edit the tree, return to the form using "Edit Family Tree".
